🌿 EcoGreen - Modern Sustainable Solutions Landing Page
A premium, high-quality, and fully responsive landing page designed for eco-friendly businesses and environmental projects. This project focuses on clean UI, modern aesthetics, and smooth user experience.

✨ Features
Modern UI/UX: Clean and professional layout using the #068066 (Teal Green) color palette.

Fully Responsive: Optimized for Mobile, Tablet, and Desktop screens.

Interactive Elements: Smooth hover effects and organized navigation.

Google Fonts Integration: Uses 'Poppins' for a sleek and readable look.

Font Awesome Icons: High-quality icons for services and social media.

🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Custom styling, Flexbox, and CSS Variables.

Google Fonts: For modern typography.
